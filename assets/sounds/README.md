## Sound Effects

Place the following .mp3 files in this directory:

| File | Description | Suggested duration |
|------|------------|-------------------|
| `typewriter-click.mp3` | Single key press sound | ~50ms |
| `paper-rustle.mp3` | Paper shuffling / page turn | ~300ms |
| `folder-snap.mp3` | Folder opening / closing snap | ~200ms |
| `vhs-static.mp3` | Brief VHS static noise | ~200ms |
| `alarm-subtle.mp3` | Subtle alert tone for contradictions | ~500ms |
| `key-click.mp3` | Mechanical keyboard key press | ~80ms |

Free sources:
- https://freesound.org (search "typewriter click", "paper rustle", etc.)
- https://pixabay.com/sound-effects/
- https://mixkit.co/free-sound-effects/
