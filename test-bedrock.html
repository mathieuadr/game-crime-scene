CORS note: AWS Bedrock Runtime endpoints are not browser-accessible (no CORS headers). If you see "Failed to fetch", the fix is to add a tiny proxy. The simplest option is to run:

npx local-cors-proxy --proxyUrl https://bedrock-runtime.us-west-2.amazonaws.com --port 8010

Then set baseUrl in config.js to http://localhost:8010/openai/v1 and keep provider as "openai" (so the url map is bypassed).
